# companion-module-renewedvision-pvp

Bitfocus Companion module for controlling Renewed Vision PVP 3.

This fork adds expanded live variables and grouped feedbacks for playlists, cues, layers, transport state, effects, transitions, target sets, blend modes, presets, and workspace status while keeping the original PVP control actions intact.

Variable examples:

```txt
$(pvp:playlist_0_cue_0_name)
$(pvp:transport_0_time_remaining_clock)
$(pvp:layer_0_opacity)
```

See `companion/HELP.md` and `LICENSE`.
